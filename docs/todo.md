# Lingo Jungle Working Todo

## Locked Decisions

- Project name: `Lingo Jungle`
- Main UI direction: bright jungle / tropical adventure
- Main frontend stack: `Next.js + TypeScript + Tailwind CSS`
- Main data/auth stack: `Supabase + Google OAuth`
- Additional diploma backend: `Java Spring Boot`

## Java Service Scope

- validate answers
- detect mistake type
- return a short grammar explanation
- select lesson content by language, level, and goal
- recommend next lessons from user progress
