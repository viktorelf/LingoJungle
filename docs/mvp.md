# Lingo Jungle MVP

## Product Goal

Create a diploma-ready language learning web app with a bright jungle identity, personalized lesson context, and simple but polished gamification.

## Languages

- English
- French

## Learning Goals

- travel
- work
- movies
- general development

## Avatars

- parrot
- fox
- panda

## Exercise Types

- multiple choice
- matching
- sentence builder
- fill in the blank

## Java Logic Features

- answer validation
- mistake classification
- short grammar hint generation
- lesson selection by goal
- progress-based lesson recommendation
